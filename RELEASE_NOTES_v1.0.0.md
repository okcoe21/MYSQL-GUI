# MySQL GUI v1.0.0 — Initial Release

Welcome to the initial public release of **MySQL GUI**!

MySQL GUI is a high-performance, secure database administration client designed for local development and professional database management, styled with a sleek terminal-core dark UI.

### Key Highlights
- **Standalone Tauri v2 Desktop App**: Light, fast, native executables for Windows, macOS, and Linux.
- **Secure Keychain Storage**: Protects your API keys/LLM configurations locally via the OS keychain (`keyring` crate).
- **Observability Dashboard**: Live telemetry charts measuring server queries, connections, and traffic.
- **Visual ER Diagrams & Query Builder**: Easily view database relations and generate queries visually.
- **Advanced Import/Export**: Execute large SQL scripts with rollback safety, and export data as SQL, CSV, or JSON.
